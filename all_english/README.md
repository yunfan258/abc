# all_english（仿扇贝单词）

       这是一款移动端的背单词网页，包括13个页面。

       首页，课程页，发现页，我的页，开始学习页，搜索页，通知信息页，复习单词页，书籍详情页，修改计划天数页，学习数据页，打卡日历页，设置页。

       主要用于背考研单词，使用了网上的mock平台，因为缺少数据，所以有些页面是静态的，使用的vue3的setup语法，运用了element-plus，scss，采用组件化设计的思想，使用axios请求淘宝的mock平台数据
